to use this library you have to add following line in dependency section of cargo.toml

`task2 = "0.1.0"`

your cargo.toml file should look like this:
```
[package]
name = "task2"
version = "0.1.0"
authors = ["muhammadtalemand <talemandkhokhar@gmail.com>"]
edition = "2018"

task1 = "0.1.0"
```
